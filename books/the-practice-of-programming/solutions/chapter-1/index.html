Chapter 1 - Style: Solutions

Exercise 1-1

Comment on the choice of names and values in the following code

    
    #define TRUE 0
    #define FALSE 1

    if ((ch = getchar()) == EOF)
      not_eof = FALSE
Using macros to define TRUE as 0 and FALSE as 1 completely goes against all idiomatic C, and indeed idiomatic computer science behavior. stdhool.h is widely used and a far better alternative to defining custom macros for boolean values.
